## 二、工程约束（所有 Task 必须遵守）

| # | 约束 | 说明 |
|---|------|------|
| C1 | 零外部依赖安装 | FFmpeg、SkiaSharp、NAudio 全部内置在项目中，用户不需要额外安装任何运行时 |
| C2 | 路径安全 | 所有文件路径操作必须处理空格和中文字符；FFmpeg 命令参数用双引号包裹路径 |
| C3 | 进程安全 | FFmpeg 子进程绑定到 Windows Job Object，主进程退出时自动终止所有子进程 |
| C4 | UI 响应性 | 所有耗时操作（>50ms）必须在后台线程执行，通过 Dispatcher 回调 UI；长操作提供取消按钮；防止"程序未响应" |
| C5 | 编译自测 | 每个 Task 完成后必须执行 Build + 运行对应单元测试，确认全部通过后方可进入下一个 Task |
| C6 | 编码安全 | 文件读写统一使用 UTF-8 编码 |
| C7 | FFmpeg 显式编解码器 | 所有 FFmpeg 命令必须通过 `-c:v` / `-c:a` 显式指定编解码器，不依赖系统默认行为 |
| C8 | 统一配置文件 | 所有配置信息收拢到 `AppConfig.json`，禁止硬编码。包括但不限于：FFmpeg 路径、默认输出目录、转换预设参数、帧缓存大小、波形分块宽度、LRU 缓存帧数、吸附阈值、Undo 栈深度、日志轮转大小/份数、水印文字内容/位置等 |
